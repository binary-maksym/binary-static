Problema com Login

Se estiver tendo dificuldades conectando-se à sua conta, verifique se os cookies estão ativados no seu browser. Este website utiliza cookies para guardar o seu login e senha (em formato encriptado) no seu PC durante a sua sessão de login.

Para ativar os cookies, siga as instruções abaixo para a versão de browser que está a utilizar.

Binary (C.R.) S.A.

Chrome 40+

  • Vá para"Menu Chrome (Personalizar e controlar Google Chrome)" abra e selecione Configurações.
  • Clique em Mostrar configurações avançadas.
  • Clique no botão Configurações de conteúdo sob o menu Privacidade.
  • Selecione Permitir a configuração de dados locais (recomendado).
  • Clique no botão Fechar para salvar as suas configurações.
Binary (IOM) Ltd

Mozilla Firefox 40+

  • Abra o menu e selecione Preferências.
  • Clique na aba Privacidade.
  • Defina "O Firefox deve" para "Lembrar histórico".
  • Clique no botão OK para salvar as suas definições.
Binary (Europe) Ltd

Safari 5.x+

  • Vá para o menu Safari e selecione Preferências.
  • Clique na aba Privacidade.
  • Defina "Bloquear cookies:" para Nunca.
  • Clique no botão X no canto superior direito para salvar e sair.
Binary (C.R.) S.A.

Opera 30+

  • Vá para o menu Ferramentas e selecione Preferências.
  • Clique no menuAvançadas.
  • Selecione Cookies na coluna esquerda.
  • Selecione Aceitar cookies.
  • Clique no botão OK para salvar e fechar.
Binary Ltd

Microsoft Internet Explorer 11+

  • Vá para o menu Ferramentas e selecione Opções de Internet.
  • Clique no menu Privacidade.
  • Clique no botão Avançadas.
  • Selecione Ignorar tratamento automático de cookies.
  • Selecione Aceitar tanto para Cookies Primários quanto para Cookies de Terceiros.
  • Clique no botão OK para salvar e fechar.
Binary Ltd

Microsoft Edge 12+

  • Vá para o menu Mais ações e selecione Definições.
  • SobDefinições avançadas, clique no botão Ver definições avançadas .
  • Na seção Cookies, selecione a opção "Não bloquear cookies".
  • Clique em qualquer sítio fora do menu para salvar e sair.
